## 5. ELEMENTI GRAFICI

### Cornici "Italiane"
- Ispirate a cartoline d'epoca
- Bordi sottili (1-2px) color pietra o terracotta
- Angoli leggermente arrotondati o con dettagli decorativi minimali
- Usate per: immagini hero, cards suite, citazioni

### Linee Decorative
- Separatori orizzontali sottili con piccolo elemento centrale (rombo, foglia stilizzata)
- Linee verticali per sidebar o timeline

### Pattern (uso leggero)
- Texture carta pregiata (overlay 5-10% opacity)
- Motivo geometrico italiano molto sottile per sezioni speciali

### Icone
- Stile: line art, stroke 1.5-2px
- Set coerente per: suite amenities, esperienze, contatti
- Possibile set custom con elementi: foglia, calice, tartufo, fiamma, lanterna
